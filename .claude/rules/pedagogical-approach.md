# Педагогический подход

## При проверке работ
- Начинай с положительных моментов в работе ученика
- Указывай на ошибки конструктивно, с объяснением правильного варианта
- Предлагай конкретные рекомендации по улучшению
- Учитывай уровень ученика (слабый/средний/сильный)
- Не допускай уничижительных формулировок

## При составлении заданий
- Предусматривай дифференциацию: базовый (60%), повышенный (30%), высокий (10%) уровни
- Включай задания на разные виды деятельности (воспроизведение, применение, анализ, творчество)
- Формулировки должны быть однозначными и понятными для возраста
- Для тестов: 4 варианта ответа, один правильный (если не указано иное)
- Избегай заданий-ловушек и двусмысленных формулировок

## При планировании уроков
- Соблюдай структуру системно-деятельностного подхода
- Предусматривай смену видов деятельности каждые 7-10 минут
- Планируй здоровьесберегающие паузы (физкультминутка, гимнастика для глаз)
- Включай работу в парах и группах
- Предусматривай этап рефлексии
- Домашнее задание: базовое для всех + дополнительное по выбору

## При оценивании
- Используй 5-балльную шкалу с чётким описанием критериев
- «5» — полное, безошибочное выполнение
- «4» — допущены 1-2 незначительные ошибки
- «3» — выполнено более 50%, допущены существенные ошибки
- «2» — выполнено менее 50%, грубые ошибки или невыполнение
- Для ОГЭ/ЕГЭ — используй официальные критерии ФИПИ
