# Математика — Предметные правила

## Структура курса по классам
- **5 класс:** Натуральные числа, Дроби (обыкновенные и десятичные), Геометрические фигуры, Измерения
- **6 класс:** Рациональные числа, Отношения и пропорции, Координатная плоскость, Симметрия
- **7 класс:** Алгебра: Линейные уравнения, Функции, Степень. Геометрия: Начала планиметрии, Треугольники
- **8 класс:** Алгебра: Квадратные уравнения, Неравенства, Квадратный корень. Геометрия: Четырёхугольники, Площади, Подобие
- **9 класс:** Алгебра: Прогрессии, Элементы статистики и теории вероятностей. Геометрия: Векторы, Окружность. Подготовка к ОГЭ
- **10 класс:** Тригонометрия, Производная, Стереометрия (базовый). Углублённый: + Комбинаторика, Комплексные числа
- **11 класс:** Интеграл, Показательная и логарифмическая функции, Тела вращения. Подготовка к ЕГЭ

## Правила оформления решений
- Дано → Найти → Решение → Ответ (для задач)
- Пошаговое решение с пояснением каждого действия
- Единицы измерения в ответе
- Для геометрии: описание чертежа (или ASCII-схема)
- Дробная черта для обыкновенных дробей, точка для десятичных

## Формат ОГЭ (9 класс)
- Часть 1 (задания 1-19): с кратким ответом
- Часть 2 (задания 20-25): с развёрнутым ответом
- Модули: Алгебра, Геометрия, Реальная математика

## Формат ЕГЭ (11 класс)
- Часть 1 (задания 1-11): с кратким ответом (базовый уровень)
- Часть 2 (задания 12-18): с развёрнутым ответом (повышенный и высокий)

## Дифференциация по уровням
- Базовый: вычисления, простые уравнения, типовые задачи
- Повышенный: составные уравнения, задачи на несколько действий, доказательства
- Высокий: нестандартные задачи, олимпиадного типа, задачи с параметром
