# Стандарты оформления документов

## План урока
- **Заголовок:** предмет, класс, тема, дата, ФИО учителя
- **Тип урока:** изучение нового / закрепление / контроль / комбинированный / урок-практикум
- **Цели:** обучающая, развивающая, воспитательная
- **Планируемые результаты:**
  - Предметные: конкретные знания и умения
  - Метапредметные: познавательные, регулятивные, коммуникативные УУД
  - Личностные: ценности, отношения
- **Оборудование:** учебник, доска, ТСО, раздаточный материал
- **Ход урока:** таблица (этап | время | деятельность учителя | деятельность учеников)
- **Домашнее задание:** базовое + по выбору

## Контрольная работа / тест
- **Заголовок:** предмет, класс, тема, вариант
- **Инструкция** для ученика (как оформлять, время выполнения)
- **Задания** с нумерацией (часть A, B, C при наличии)
- **Система оценивания:** баллы за каждое задание → итоговый балл → оценка
- **Ключ ответов** (в отдельном файле или разделе)

## Тематическое планирование (КТП)
- **Таблица:** № п/п | Тема урока | Кол-во часов | Дата план | Дата факт | Тип урока | Виды контроля | Примечание
- **Итого** часов за период
- **Привязка** к разделам ФГОС и примерной программы

## Общие правила оформления
- Markdown: заголовки ##, таблицы, списки
- Кириллица для всех надписей
- Числа записывать цифрами (5 класс, 45 минут)
- Даты: ДД.ММ.ГГГГ
