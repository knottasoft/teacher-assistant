# Генерация задач по математике

- **Класс:** $ARGUMENTS[0]
- **Тема:** $ARGUMENTS[1]
- **Количество:** $ARGUMENTS[2] (по умолчанию: 10)

## Требования

1. Задачи трёх уровней сложности:
   - **Базовый** (60%) — прямое применение формул/правил
   - **Повышенный** (30%) — задачи в 2-3 действия, нестандартная формулировка
   - **Высокий** (10%) — комбинированные задачи, задачи с параметром

2. Для каждой задачи:
   - Условие (чёткая формулировка)
   - Подробное решение (с пояснением каждого шага)
   - Ответ

3. Правила оформления:
   - Обыкновенные дроби через дробную черту
   - Знак умножения: ×  или ·
   - Для геометрии — описание чертежа
   - Единицы измерения в ответе

4. Учитывай раздел: алгебра / геометрия / теория вероятностей

5. Для 9 класса — включай задачи в формате ОГЭ
6. Для 11 класса — включай задачи в формате ЕГЭ

7. Сохрани:
   - `задачи_математика_[класс]_[тема].md` — условия
   - `задачи_математика_[класс]_[тема]_решения.md` — решения с ответами
