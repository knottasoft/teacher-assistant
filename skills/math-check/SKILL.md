# Проверка решения по математике

Проверь решение: $ARGUMENTS

## Порядок проверки

1. Прочитай решение ученика
2. Если не указано — спроси: класс, тема, условие задачи
3. Проверь:
   - Правильность выбранного метода решения
   - Корректность каждого шага
   - Вычислительные ошибки
   - Правильность ответа
   - Оформление (Дано → Найти → Решение → Ответ)
4. Оцени решение:
   - **Верно** — решение полностью правильное
   - **Частично верно** — верный подход, но есть ошибки
   - **Неверно** — принципиальная ошибка в подходе
5. При наличии ошибок:
   - Укажи, где именно допущена ошибка
   - Объясни, почему это ошибка
   - Покажи правильное решение пошагово
6. Выстави оценку, если это контрольная/самостоятельная работа
