# Инструкция к лабораторной работе

- **Класс:** $ARGUMENTS[0]
- **Тема:** $ARGUMENTS[1]

## Структура

Создай полную инструкцию к лабораторной работе:

1. **Лабораторная работа №[N]**
   - Название
   - Дата

2. **Цель работы** — что измерить/определить/проверить

3. **Оборудование** — из стандартного школьного набора:
   - Мензурка, линейка, весы, динамометр и т.д.
   - Указать точность приборов

4. **Теоретическая часть** (краткая):
   - Основные формулы
   - Физический смысл величин

5. **Порядок выполнения** (пошаговый):
   - Шаг 1: ...
   - Шаг 2: ...
   - (подробно, чтобы ученик мог выполнить самостоятельно)

6. **Таблица для записи результатов:**

| № опыта | Величина 1 | Величина 2 | Результат |
|---------|-----------|-----------|----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

7. **Обработка результатов:**
   - Формулы для расчёта
   - Расчёт погрешностей (для 10-11 класса)

8. **Контрольные вопросы** (5-7 вопросов)

9. **Вывод** — шаблон для ученика
