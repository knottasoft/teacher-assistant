# Темы сочинений

- **Класс:** $ARGUMENTS[0]
- **Произведение/направление:** $ARGUMENTS[1]

## Структура

1. Предложи 5-7 тем сочинений разной направленности:
   - По конкретному произведению (анализ образа, проблемы)
   - Сравнительный анализ (два героя, два произведения)
   - Проблемный вопрос (дискуссионная формулировка)
   - Свободная тема по направлению

2. Для каждой темы:
   - **Формулировка** (точная, однозначная)
   - **Тип:** анализ образа / сравнение / проблемный вопрос / свободная
   - **Рекомендуемые произведения** для аргументации
   - **Ключевые тезисы** (3-4 пункта для плана сочинения)
   - **Критерии оценки** (на что обратить внимание)

3. Для 9 класса — учитывай формат ОГЭ (задание 9.1/9.2/9.3)
4. Для 11 класса — ориентируйся на направления итогового сочинения

5. Сохрани: `темы_сочинений_[класс]_[произведение].md`

Направления итогового сочинения: [ege-essay-directions.md](ege-essay-directions.md)
