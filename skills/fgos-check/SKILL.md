# Проверка соответствия ФГОС

Проверь документ на соответствие ФГОС: $ARGUMENTS

## Порядок проверки

1. Прочитай документ
2. Определи: предмет, класс, тип документа (план урока / КТП / рабочая программа)
3. Загрузи требования ФГОС через `fgos_lookup`
4. Проверь по чек-листу:

### Для плана урока:
- ✅ Указан тип урока
- ✅ Сформулированы цели (обучающая, развивающая, воспитательная)
- ✅ Указаны планируемые результаты (предметные, метапредметные, личностные)
- ✅ Отражено формирование УУД
- ✅ Реализован системно-деятельностный подход
- ✅ Предусмотрена дифференциация
- ✅ Есть этап рефлексии
- ✅ Корректность формулировок по терминологии ФГОС
- ✅ Содержание соответствует программе для данного класса

### Для КТП:
- ✅ Указано количество часов (соответствует ФГОС)
- ✅ Все разделы программы представлены
- ✅ Контрольные работы запланированы
- ✅ Есть привязка к планируемым результатам
- ✅ Виды контроля разнообразны

5. Выдай отчёт:
   - **Соответствует** / **Частично соответствует** / **Не соответствует**
   - Список замечаний с указанием пункта ФГОС
   - Рекомендации по доработке
   - Общая оценка качества документа
