# Формат экзаменационных работ по математике

## ОГЭ по математике (9 класс)

**Время:** 3 часа 55 минут (235 мин)
**Заданий:** 25

| Часть | Задания | Тип ответа | Баллы |
|-------|---------|-----------|-------|
| 1 | 1-19 | Краткий ответ | по 1б = 19б |
| 2 | 20-25 | Развёрнутое решение | по 2б = 12б |

**Модули:** Алгебра (задания 1-14, 20-22), Геометрия (15-19, 23-25)
**Итого:** 31 балл
**Обязательно:** минимум 2 балла по модулю «Геометрия»

| Оценка | Баллы |
|--------|-------|
| 5 | 22-31 |
| 4 | 15-21 |
| 3 | 8-14 |
| 2 | 0-7 |
