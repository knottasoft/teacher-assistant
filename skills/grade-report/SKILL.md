# Анализ успеваемости

Данные: $ARGUMENTS

## Инструкции

1. Загрузи данные об оценках:
   - Файл CSV: `ФИО,оценка1,оценка2,...`
   - Файл JSON: `[{"student": "ФИО", "grades": [5,4,3...]}]`
   - Или попроси ввести вручную
2. Используй `grade_analytics` для расчёта статистики
3. Сформируй отчёт, включающий:

### Общая статистика
- Средний балл по классу
- Качество знаний (% учеников на 4 и 5)
- Успеваемость (% учеников без двоек)
- Распределение оценок (сколько 5, 4, 3, 2)

### Индивидуальный анализ
- **Ученики, нуждающиеся в поддержке** (средний балл < 3.0)
- **Ученики с высоким потенциалом** (средний балл > 4.5)
- Динамика по периодам (если есть данные за несколько периодов)

### Рекомендации
- Для учителя: на какие темы обратить внимание
- Для отстающих: индивидуальная работа
- Для сильных учеников: олимпиады, углублённые задания

4. Сохрани отчёт в файл `отчёт_успеваемость_[класс].md`

Шаблоны отчётов: [report-templates.md](report-templates.md)
