# Проверка работы ученика

Проверь работу ученика: $ARGUMENTS

## Порядок проверки

1. Прочитай работу ученика (файл или текст)
2. Если не указано — спроси: предмет, класс, тема, тип работы (диктант/тест/сочинение/задачи)
3. Проверь каждое задание:
   - Правильные ответы — отметь ✓
   - Ошибки — отметь ✗ с пояснением правильного ответа
   - Частично верные — укажи, что верно и что нет
4. Подсчитай баллы
5. Выстави оценку по 5-балльной шкале (или по критериям ОГЭ/ЕГЭ, если указано)
6. Напиши развёрнутый комментарий:
   - **Что сделано хорошо** (начни с положительного)
   - **Типичные ошибки** (с объяснением правил)
   - **Рекомендации по улучшению**
7. Сохрани результат проверки в файл `проверка_[имя_ученика]_[тема].md`
