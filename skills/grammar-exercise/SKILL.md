# Упражнение по грамматике

- **Класс:** $ARGUMENTS[0]
- **Тема:** $ARGUMENTS[1]

## Инструкции

1. Определи тип упражнения по теме:
   - Вставить пропущенные буквы
   - Расставить знаки препинания
   - Разбор (морфемный / морфологический / синтаксический)
   - Найти и исправить ошибки
   - Образовать формы слова
   - Составить предложения по схеме
2. Создай 8-12 заданий нарастающей сложности
3. Используй разнообразные примеры (не однотипные)
4. Создай ключ с ответами и пояснениями правил
5. Сохрани:
   - `упражнение_[тема]_[класс].md` — для ученика
   - `упражнение_[тема]_[класс]_ответы.md` — ключ для учителя
