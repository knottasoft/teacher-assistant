# Создание диктанта

Создай диктант:
- **Класс:** $ARGUMENTS[0]
- **Тема/орфограммы:** $ARGUMENTS[1]

## Требования

1. **Объём текста** по нормам:
   - 5 класс: 90-100 слов
   - 6 класс: 100-110 слов
   - 7 класс: 110-120 слов
   - 8 класс: 120-150 слов
   - 9 класс: 150-170 слов
   - 10-11 класс: 170-200 слов

2. **Содержание текста:**
   - Связный, тематически цельный текст (повествование/описание/рассуждение)
   - Содержит целевые орфограммы/пунктограммы (не менее 10-15 случаев)
   - Не содержит незнакомых ученикам слов
   - Соответствует возрасту по тематике

3. **Грамматическое задание** (2-3 задания):
   - Морфемный/словообразовательный разбор
   - Морфологический разбор
   - Синтаксический разбор предложения
   - Или другие задания по теме

4. **Лист для учителя:**
   - Текст с выделенными орфограммами (подчёркнуты или выделены)
   - Перечень контролируемых правил
   - Нормы оценивания диктанта (по классу)
   - Ответы к грамматическому заданию

5. Сохрани два файла:
   - `диктант_[класс]_[тема].md` — для диктовки
   - `диктант_[класс]_[тема]_учитель.md` — лист для учителя

Уровни сложности: [difficulty-levels.md](difficulty-levels.md)

## Self-validation перед завершением

Перед тем как сообщить пользователю «готово», проверь, что выходной файл содержит:

- [x] **Класс** (например, «5 класс», «8 класс»)
- [x] **Предмет** (русский язык, математика, физика, литература)
- [x] **Привязка к нормативной базе:** ФГОС (приказ Минпросвещения №287/2021 для ООО, приказ №413/2012 в действующей редакции для СОО), ФОП (№370 или №371 от 2023), либо ФРП от 2024
- [x] **Планируемые результаты** или **УУД** (для планов уроков, заданий, КТП)
- [x] **Тип документа** в заголовке (план урока / задание / тест / отчёт / КТП)

Если чего-то не хватает — допиши до возврата управления, не возвращай учителю недооформленный документ. Это правило заменяет общий PostToolUse-hook в плагине и относится только к выходу этого skill'а.
