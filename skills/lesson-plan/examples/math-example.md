# План урока: Математика (Алгебра), 8 класс

## Тема: Квадратные уравнения. Формула корней квадратного уравнения

**Тип урока:** Комбинированный
**Время:** 45 минут

### Цели урока
- **Обучающая:** Вывести формулу корней квадратного уравнения, научить применять её для решения уравнений
- **Развивающая:** Развивать логическое мышление, умение выполнять алгебраические преобразования
- **Воспитательная:** Воспитывать аккуратность при вычислениях, настойчивость в достижении результата

### Планируемые результаты
- **Предметные:** Знать формулу дискриминанта и корней квадратного уравнения, уметь применять формулу для решения уравнений
- **Метапредметные:** Умение строить логическую цепочку рассуждений, выполнять преобразования по алгоритму
- **Личностные:** Понимание значимости математических знаний для решения практических задач

### Оборудование
Учебник, доска, калькулятор (по необходимости), карточки с заданиями

### Оформление решения (образец)

```
x² − 5x + 6 = 0
a = 1, b = −5, c = 6
D = b² − 4ac = 25 − 24 = 1 > 0 — два корня
x₁ = (5 + 1) / 2 = 3
x₂ = (5 − 1) / 2 = 2
Ответ: x = 2; x = 3
```
