# Справочник основных формул по физике

## Механика (7, 9-10 класс)
- Скорость: v = s/t
- Равноускоренное движение: s = v₀t + at²/2, v = v₀ + at
- Второй закон Ньютона: F = ma
- Сила тяжести: F = mg
- Сила упругости (Гука): F = kΔl
- Сила трения: F = μN
- Импульс: p = mv
- Закон сохранения импульса: m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
- Работа: A = Fs·cos α
- Мощность: P = A/t = Fv
- Кинетическая энергия: Eк = mv²/2
- Потенциальная энергия: Eп = mgh
- Закон сохранения энергии: Eк₁ + Eп₁ = Eк₂ + Eп₂

## Давление (7 класс)
- Давление: p = F/S
- Давление жидкости: p = ρgh
- Сила Архимеда: Fа = ρ_ж·g·V_тела
- Закон Паскаля: давление передаётся одинаково во все стороны

## Тепловые явления (8 класс)
- Количество теплоты: Q = cmΔt
- Теплота плавления: Q = λm
- Теплота парообразования: Q = Lm
- Теплота сгорания: Q = qm
- Уравнение теплового баланса: Q_отд = Q_получ

## Электричество (8, 10 класс)
- Закон Ома: I = U/R
- Сопротивление: R = ρl/S
- Последовательное: R = R₁ + R₂, I = const, U = U₁ + U₂
- Параллельное: 1/R = 1/R₁ + 1/R₂, U = const, I = I₁ + I₂
- Мощность тока: P = UI = I²R = U²/R
- Работа тока: A = UIt
- Закон Джоуля-Ленца: Q = I²Rt
- Закон Кулона: F = k·q₁q₂/r²

## Оптика (8, 11 класс)
- Закон отражения: угол падения = угол отражения
- Закон преломления: n₁·sin α = n₂·sin β
- Формула тонкой линзы: 1/F = 1/d + 1/f
- Оптическая сила: D = 1/F

## Колебания и волны (9, 11 класс)
- Период: T = 1/ν
- Длина волны: λ = vT
- Период маятника: T = 2π√(l/g)
- Период пружинного: T = 2π√(m/k)

## Квантовая физика (11 класс)
- Энергия фотона: E = hν
- Фотоэффект: hν = Aвых + Eк_max
- Энергия связи: E = Δm·c²
