# Учитель — плагин для Claude Code

Ассистент для преподавателей средней и старшей школы (5–11 классы). Помогает с ежедневными задачами: создание планов уроков, составление заданий, проверка работ учеников, тематическое планирование, анализ успеваемости.

Все материалы формируются в соответствии с **ФГОС ООО** (приказ №287 от 31.05.2021) и **ФГОС СОО** (приказ №732 от 12.08.2022).

**Предметы:** Русский язык, Математика, Физика, Литература

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/knottasoft/teacher-assistant.git
cd teacher-assistant
```

### 2. Собрать MCP-сервер

```bash
cd mcp-server
npm install
npm run build
cd ..
```

### 3. Открыть в Claude Code

Откройте директорию `teacher-assistant` в Claude Code. Плагин подключится автоматически — CLAUDE.md, правила, навыки и MCP-сервер будут загружены при старте сессии.

## Команды

### Основные

| Команда | Описание | Пример |
|---------|----------|--------|
| `/lesson-plan` | Создать план урока по ФГОС | `/lesson-plan математика 7 "Формулы сокращённого умножения"` |
| `/assignment` | Создать задание (тест, ДЗ, контрольная) | `/assignment физика 8 "Закон Ома" тест` |
| `/check-work` | Проверить работу ученика | `/check-work работа_ученика.md` |
| `/thematic-plan` | Календарно-тематическое планирование | `/thematic-plan русский 6 год` |
| `/fgos-check` | Проверить документ на соответствие ФГОС | `/fgos-check план_урока.md` |
| `/grade-report` | Анализ успеваемости класса | `/grade-report оценки_7б.csv` |
| `/export-doc` | Экспорт в DOCX | `/export-doc план_урока.md` |
| `/import-template` | Импорт шаблона школы | `/import-template шаблон_ктп.md thematic-plan` |

## Форматы вывода

- **Markdown** — основной формат, готов к просмотру в Claude Code
- **DOCX** — экспорт через команду `/export-doc`, формат для печати и документооборота (Times New Roman 12pt, поля: 3см лево / 1.5см право / 2см верх-низ)

## Импорт шаблонов

У каждой школы могут быть свои шаблоны документов. Используйте `/import-template` для загрузки:

```
/import-template шаблон_ктп_школа15.md thematic-plan
```

Поддерживаемые типы: `lesson-plan`, `test`, `thematic-plan`, `report`, `assignment`, `lab-work`

Импортированные шаблоны сохраняются в `user-data/templates/` и автоматически используются при создании документов.

## MCP-инструменты

Плагин предоставляет MCP-сервер со следующими инструментами:

| Инструмент | Описание |
|-----------|----------|
| `fgos_lookup` | Поиск по базе ФГОС: разделы, темы, планируемые результаты, УУД |
| `export_docx` | Конвертация Markdown → DOCX |
| `import_template` | Импорт и анализ пользовательского шаблона |
| `hours_calculator` | Расчёт учебных часов за период (четверть / полугодие / год) |
| `grade_analytics` | Анализ успеваемости: средний балл, качество, «группа риска» |

## Система оценивания

Используется 5-балльная шкала по российским стандартам:

| Оценка | % выполнения | Описание |
|--------|-------------|----------|
| 5 (отлично) | 90–100% | Задание выполнено полностью, без ошибок |
| 4 (хорошо) | 70–89% | Выполнено с незначительными ошибками |
| 3 (удовлетворительно) | 50–69% | Выполнена обязательная часть |
| 2 (неудовлетворительно) | < 50% | Не освоен базовый уровень |

## Дифференциация заданий

При создании заданий автоматически используется принцип дифференциации:

- **Базовый уровень** — 60% заданий (обязательный минимум)
- **Повышенный уровень** — 30% заданий (углублённое изучение)
- **Высокий уровень** — 10% заданий (олимпиадный / исследовательский)
